IPU Vault 📚

A modern platform for accessing Previous Year Question Papers (PYQs) for IP University students.

🌐 Live Website:
ipuvault.vercel.app

 Features: 
📂 Branch-wise PYQ collection
📚 Semester & subject organization
⚡ Fast and responsive UI
🌙 Clean modern interface
🔍 Easy navigation for students
📱 Mobile-friendly design
☁️ Cloud-hosted with fast delivery
📊 Integrated Vercel Analytics

⚙️ Installation & Setup

Clone the repository:

git clone <your-repository-url>

Navigate to project folder:

cd ipu-vault

Install dependencies:

npm install

Run development server:

npm run dev
📊 Analytics

This project uses:

Vercel Analytics for visitor tracking
Performance monitoring
Traffic insights

🎯 Purpose

IPU Vault was built to simplify access to PYQs for students by providing a centralized and organized platform instead of scattered PDFs and links.
